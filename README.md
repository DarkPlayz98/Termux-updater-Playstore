# Termux Updater Playstore
A custom script designed to revive the outdated Play Store version of Termux. 

Instead of downloading a whole new app, this script hijacks the internal package manager, connects it to the modern F-Droid/GitHub servers, fixes the broken security keys, and updates all your internal tools to the absolute latest versions.

## How to Install and Run
Run these commands inside your Termux app:

1. `git clone https://github.com/DarkPlayz98/termux-updater.git`
2. `cd termux-updater`
3. `chmod +x update.sh`
4. `./update.sh`
